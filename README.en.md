## Arcaea Scores

A tool that uses Android Root access to read local Arcaea save data and calculate Best 30 results.

### Download

Download the latest ArcaeaScores APK from [Releases](https://github.com/Rosemoe/ArcaeaScores/releases/latest).

### Features

* Automatically extracts local play records
* Calculates Best 30 / Max PTT
* Displays chart constants, play potential, judgment data, clear status, and lost score for each record
* Filters and sorts scores
* Automatically extracts and displays song artwork from the game
* Updates Arcaea chart-constant data online, synchronized with Yurisaki Bot
