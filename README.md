## Arcaea Scores
使用 Android Root 权限获取本地 Arcaea 存档，并计算 Best30 的工具
### 下载
从 [Releases](https://github.com/Rosemoe/ArcaeaScores/releases/latest) 下载最新的 ArcaeaScores APK 文件。
### 功能
* 自动提取本地游玩记录
* 计算 Best30/MaxPtt
* 展示记录的谱面定数/游玩潜力值/判定数据/通关状态/失分
* 筛选、排序成绩
* 自动从游戏提取曲绘并展示
* 在线更新 Arcaea 定数数据，与 Yurisaki Bot 保持同步
